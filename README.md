# UP2Controller


**2018 ESDC Best Prize Project — Omni-directional SLAM-based Control System using Intel UP²**

UP2Controller is a control system designed for real-time autonomous motion using Omni-directional SLAM and FPGA-based hardware control. This project was developed on the Intel UP² board and integrates computer vision, motor control, and PCAN-based communication.

---

## Features

- Real-time motor control through FPGA
- Visual motion detection using OpenCV
- PCAN communication with low-level controllers
- Modular multithreaded architecture
- Designed for embedded robotics on the UP² platform
